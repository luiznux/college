Ambiente computacional necessário para desenvolvimento da disciplina. 

Os trabalhos serão desenvolvidos em Python.

O aconselhável é instalar a distribuição Anaconda, com Python 3 como versão root. 
(Caso queira, depois você pode instalar, via Anaconda, Python 2.7 num outro environment.)

Isso vai instalar o ambiente mínimo necessário:
- Python
- NumPy 
- Matplotlib 
- pandas
